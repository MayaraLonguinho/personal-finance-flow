# Prompt 12: Relatórios

Implemente relatórios por período:

1. **Filtros**:
   - Data início, data fim (YYYY-MM-DD)
   - Valida que data início ≤ data fim

2. **Conteúdo do relatório**:
   - Resumo financeiro do período
   - Gastos por categoria no período
   - Lista de transações do período
   - Resumo investimentos (atual)
   - Botão de impressão
