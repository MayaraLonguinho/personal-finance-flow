# Prompt 21: Zip de Entrega

Preparar zip de entrega final:

1. **Antes do zip**:
   - Certifique-se que toda documentação em docs/ está organizada conforme a estrutura solicitada
   - Atualize docs/README.md com índice completo
   - Verifique que não há dados sensíveis no repositório

2. **Criar zip**:
   - Arquivo com nome apropriado
   - Inclui todos os arquivos necessários
   - **NÃO INCLUA**:
     - .venv/
     - .env
     - uploads/
     - __pycache__/
     - *.pyc
     - Qualquer arquivo temporário

3. **Validação**:
   - Extraia o zip em uma pasta nova
   - Teste que consegue configurar e rodar seguindo o README principal
