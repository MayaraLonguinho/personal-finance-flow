# Prompt 09: Dashboard

Crie o dashboard principal:

1. **Cards de métricas**:
   - Total entradas, saídas, investimentos
   - Saldo (entradas - saídas - investimentos)
   - Cards visíveis configuráveis

2. **Gráficos com Chart.js**:
   - Gastos por categoria (pizza/rosca)
   - Evolução do saldo (linha)
   - Entradas vs saídas mensais (barras)

3. **Outras seções**:
   - Transações recentes confirmadas (quantidade configurável)
   - Resumo meta ativa
   - Resumo investimentos
   - Insights

4. Aplica tema, moeda e formato de data do usuário.
