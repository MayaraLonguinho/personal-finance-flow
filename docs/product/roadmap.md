# Roadmap de Features

## ✅ Concluídas

| Feature | Descrição |
|---|---|
| Autenticação | Cadastro, login, logout, sessões seguras |
| Dashboard Principal | Cards, gráficos, transações recentes |
| CRUD Transações | Criar, editar, excluir, filtrar |
| Importação CSV | Pipeline ETL completo com deduplicação |
| Categorização Automática | Regras por palavras-chave |
| Gestão Categorias | CRUD, categorias padrão |
| Metas Financeiras | Progresso, status ativa/concluída/cancelada |
| Investimentos | Carteira, rentabilidade, resumo |
| Relatórios | Por período, impressão |
| Assistente Híbrido | OpenAI + fallback local |
| Configurações | Temas, moedas, formatos, preferências |
| Vault Brain | Documentação técnica no brain/ |
| Skill | Skill de análise de CSV em skills/ |
| MCP | Servidor MCP em mcp/ |

## 🔄 Próximos (Ideias)

- Budget/Orçamento mensal por categoria
- Notificações e alertas
- Importação de mais formatos (OFX, QIF)
- Backup automático
- Geração nativa de PDF sem depender do diálogo de impressão do navegador
- Múltiplas metas simultâneas
- Tags em transações
