# PRD 16: Skill (Skill Formalizada)

## Objetivo

Criar uma skill formalizada para análise de CSVs financeiros, seguindo o padrão de skills.

## Estrutura da Skill

```
skills/
└── financial-csv-analyzer/
    ├── SKILL.md
    └── (outros arquivos se necessários)
```

## Conteúdo do SKILL.md

- Descrição da skill
- Objetivos
- Capacidades
- Uso com exemplos
- Integração com o projeto existente (referência para `src/extract.py`, `src/transform.py`, etc., sem duplicar código)
- Instruções de como usar

## Critérios de Aceitação

- [ ] Pasta `skills/financial-csv-analyzer/` existe
- [ ] `SKILL.md` completo existe
- [ ] Não há duplicação de código do pipeline principal
