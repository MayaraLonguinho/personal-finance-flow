# Prompt 01: Estrutura Inicial

Vamos criar um aplicativo de finanças pessoais usando Flask, MySQL e Pandas. Primeiro, explore o diretório atual para entender o que já existe. Crie um plano de desenvolvimento passo a passo com visão geral do produto, requisitos, arquitetura, e roadmap. Inclua:

1. Visão do produto (qual problema resolve, para quem, value proposition)
2. Requisitos gerais (funcionais e não funcionais)
3. Roadmap com 5-7 features principais
4. Arquitetura técnica sugerida
5. Modelo de dados inicial

Use o que já existir no projeto como base. Não crie código ainda - apenas planejamento.
