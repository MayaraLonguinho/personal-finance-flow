# Prompt 13: Investimentos

Implemente gestão de investimentos:

1. **CRUD**:
   - Campos obrigatórios: nome, tipo, valor aplicado (>0), valor atual (≥0), data de aplicação
   - Campo opcional: data de vencimento (não pode ser anterior à aplicação)
   - Status: ativo, resgatado, cancelado

2. **Resumo da carteira**:
   - Total investido (ativos)
   - Valor atual total (ativos)
   - Resultado (valor atual - aplicado)
   - Rentabilidade percentual
   - Quantidade por tipo

3. Visualiza resumo no dashboard.
