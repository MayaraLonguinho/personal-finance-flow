# Prompt 11: Metas

Implemente gestão de metas financeiras:

1. **CRUD**:
   - Campos: título, valor alvo (>0), valor atual (≥0), status (ativa/concluída/cancelada)
   - Apenas uma meta "ativa" por vez

2. **Cálculos**:
   - Progresso percentual (teto de 100%)
   - Valor restante (piso de 0)

3. **Visualização**:
   - Mostra resumo da meta ativa no dashboard
