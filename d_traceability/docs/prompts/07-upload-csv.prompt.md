# Prompt 07: Upload CSV

Implemente interface para upload de CSV:

1. Área de drop/upload no dashboard
2. Valida arquivo: extensão .csv, não vazio
3. Armazena temporariamente em uploads/
4. Chama pipeline ETL
5. Mostra relatório de resultado ao usuário: total recebidas, importadas, ignoradas, categorizadas
6. Limpa arquivo temporário após processamento
